# Iterary

**A free, browser-based toolkit for prototyping tabletop and board games.**

No signup, no install — just open [iterary.com](https://www.iterary.com) and start building.

## What you can build

- **Board Crates** — grid-based boards with drag-and-drop pieces that snap to cells
- **Card Decks** — draw, shuffle, discard with multiple areas (hand, deck, discard pile)
- **Dice** — custom dice with any number of faces and roll tracking
- **Player Areas** — organize components per player with nested crates
- **Notes & Lists** — checklists, counters, text notes for rules and tracking
- **Loggers** — event logs to track game actions
